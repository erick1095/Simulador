Crear un sistema de geolocalizacion.

El sistema debera ser capaz de localizar maletines en tiempo real,
Poder crear usuarios para que se puedan loguear y cada usuario tendra 
sus propios maletines los cuales los podra estar monitoreando,
El usuario podra crear rutas de donde saldra el maletin y a donde se dirige,
El usuario podra crear viajes los cuales necesitaran una ruta para saber por donde se iran,
El usuario podra registrar a sus clientes a los cuales se les aran los viajes,

hacer base de datos de la aplicacion con mongo db
conectar aplicacion con base de datos
hacer que un maletin se pueda geolocalizar por medio de un gps en tiempo real
graficar todo lo necesario para que se mira graficamente lo que esta sucediendo
